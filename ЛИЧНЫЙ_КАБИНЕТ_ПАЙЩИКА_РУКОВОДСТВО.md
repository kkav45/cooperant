## 3. ПОШАГОВЫЕ СЦЕНАРИИ

### Сценарий 1: Подать заявление на вступление

**Цель:** Вступить в новый кооператив

```
1. Откройте вкладку «🔍 Поиск кооператива»

2. Введите название или выберите регион:
   ┌────────────────────────────────────────┐
   │ [Введите название, ИНН или регион __]  │
   │ Регион: [Все ▼]  Тип: [Все ▼]          │
   └────────────────────────────────────────┘

3. Найдите кооператив в списке:
   ┌────────────────────────────────────────┐
   │ 🏦 КПК "Первый"                         │
   │    📍 Москва | 👥 150 пайщиков         │
   │    [📋 Подробнее] [📨 Подать заявление]│
   └────────────────────────────────────────┘

4. Нажмите «Подать заявление»

5. Заполните форму:
   • ФИО
   • Дата рождения
   • Телефон
   • Email
   • Адрес
   • Сумма пая (мин. 10 000 ₽)
   • ☑️ Согласие с уставом

6. Нажмите «Отправить заявление»

7. Перейдите во вкладку «📨 Мои заявления»
   Статус: «⏳ На рассмотрении»
```

**Что дальше:**

- Ожидайте уведомления на email (1-3 дня)
- Проверяйте статус в разделе «Мои заявления»
- После одобрения — оплатите вступительный взнос

---

### Сценарий 2: Синхронизировать данные

**Цель:** Сверить данные с кооперативом

```
1. Откройте вкладку «📋 Мои кооперативы»

2. Найдите нужный кооператив:
   ┌────────────────────────────────────────┐
   │ 🏦 КПК "Первый"              ✅ Активный│
   │    💰 Внесено: 100 000 ₽               │
   │    🔄 Синхронизировано: 2 часа назад   │
   │    [📊 Подробнее] [🔄 Синхронизировать]│
   └────────────────────────────────────────┘

3. Нажмите «🔄 Синхронизировать»

4. Дождитесь завершения:
   ┌────────────────────────────────────────┐
   │ 🔄 Синхронизация...                    │
   └────────────────────────────────────────┘

5. Результат:
   
   ✅ Успешно:
   «Синхронизировано: только что»
   
   ⚠️ Есть расхождения:
   Откроется окно с расхождениями
```

**Если найдены расхождения:**

```
┌────────────────────────────────────────────┐
│ ⚠️ Расхождения в данных                    │
├────────────────────────────────────────────┤
│ Поле       │ У вас       │ В кооперативе  │
│────────────│─────────────│────────────────│
│ Телефон    │ +7900111... │ +7900333...    │
│ Email      │ old@mail.ru │ new@mail.ru    │
├────────────────────────────────────────────┤
│ Что делать?                                │
│                                            │
│ [✅ Принять версию кооператива]            │
│ [📝 Запросить исправление]                 │
│ [⏭️ Игнорировать]                          │
└────────────────────────────────────────────┘
```

**Варианты действий:**

| Действие | Результат |
|----------|-----------|
| **Принять версию кооператива** | Ваши данные обновятся |
| **Запросить исправление** | Отправится запрос в кооператив |
| **Игнорировать** | Ничего не изменится |

## 4. ЧАСТЫЕ ВОПРОСЫ

### ❓ Как узнать, в каких кооперативах я состою?

**Ответ:** Откройте вкладку **«📋 Мои кооперативы»**. Там показаны все кооперативы, где вы являетесь пайщиком.

---

### ❓ Как найти кооператив для вступления?

**Ответ:** 

1. Откройте вкладку **«🔍 Поиск кооператива»**
2. Введите название, ИНН или регион
3. Используйте фильтры по типу и региону
4. Изучите карточку кооператива
5. Нажмите «Подать заявление»

---

### ❓ Что делать, если данные не синхронизируются?

**Ответ:**

1. Проверьте интернет-соединение
2. Попробуйте ещё раз через 5 минут
3. Если ошибка повторяется — напишите в поддержку кооператива

---

### ❓ Как обновить контактные данные?

**Ответ:**

1. Откройте профиль (нажмите на аватар 👤)
2. Нажмите «Редактировать профиль»
3. Измените данные
4. Сохраните

**Важно:** Изменения потребуют подтверждения от кооператива.

---

### ❓ Как получить справку о взносах?

**Ответ:**

1. Откройте карточку кооператива («📊 Подробнее»)
2. Найдите раздел «📄 Документы»
3. Нажмите «Запросить справку»
4. Выберите тип справки
5. Справка будет доступна через 1-3 дня

---

### ❓ Что такое расхождения в данных?

**Ответ:** Расхождения возникают, когда данные в вашем личном кабинете отличаются от данных в реестре кооператива.

**Примеры:**

| Поле | У вас | В кооперативе | Причина |
|------|-------|---------------|---------|
| Телефон | +79001112222 | +79003334444 | Вы сменили номер |
| Адрес | ул. Мира, 10 | пр. Ленина, 5 | Переезд |
| Сумма | 100 000 ₽ | 105 000 ₽ | Не синхронизировано |

---

### ❓ Как часто синхронизировать данные?

**Ответ:** 

- **Автоматически:** каждые 5 минут (в фоне)
- **Вручную:** при входе в личный кабинет
- **Перед важными действиями:** перед собранием, перед выходом

---

### ❓ Что делать, если забыл пароль?

**Ответ:** Пароль не нужен — используется авторизация через Яндекс.Диск.

Просто нажмите «Войти через Яндекс» ещё раз.

## 5. ФАЙЛЫ

### Основные файлы

| Файл | Назначение |
|------|------------|
| `member-cabinet.html` | Личный кабинет пайщика (главная страница) |
| `member-app.js` | Логика работы приложения |
| `member-auth-callback.html` | Обработка авторизации Яндекс |

### Структура данных на Яндекс.Диске

```
Яндекс.Диск пайщика:
/
└── КООПЕРАНТ/
    └── member_data/
        ├── profile.json              ← Профиль пайщика
        └── cooperatives/
            ├── coop_001/
            │   └── data.json         ← Данные по кооперативу 1
            └── coop_002/
                └── data.json         ← Данные по кооперативу 2

Яндекс.Диск кооператива:
/
└── КООПЕРАНТ/
    ├── members.json                  ← Реестр пайщиков
    ├── payments.json                 ← Взносы
    └── reports/                      ← Отчёты
```

### Глобальные функции (для разработчиков)

```javascript
// Доступны в консоли (F12)

// Инициализация
MemberApp.init()

// Авторизация
MemberApp.login()
MemberApp.logout()

// Синхронизация
MemberApp.sync('coop_001')      // Синхронизировать с одним
MemberApp.syncAll()             // Синхронизировать все

// Заявления
MemberApp.submitApplication('coop_001', data)
MemberApp.cancelApplication('app_001')

// Данные
MemberApp.state.profile         // Профиль
MemberApp.state.memberships     // Членства
MemberApp.state.applications    // Заявления
```
