# Полный перечень функций информационной системы учета пайщиков кооператива

## 1. Общие функции

### 1.1 Навигация по системе
- Главная страница с общей статистикой
- Навигационное меню для перехода между разделами
- Страницы: Пайщики, Паевые взносы, Бухгалтерия, Документы, Заявления, Заседания, Отчеты

### 1.2 Управление данными
- Сохранение данных в файл (JSON)
- Загрузка данных из файла
- Автоматическое сохранение при изменениях
- Резервное копирование данных

## 2. Модуль учета пайщиков

### 2.1 Добавление пайщика
- Ввод ФИО, даты вступления, контактов, адреса
- Выбор статуса пайщика
- Добавление примечаний
- Автоматическое присвоение ID

### 2.2 Редактирование пайщика
- Изменение личных данных
- Изменение статуса
- Обновление контактной информации
- Редактирование примечаний

### 2.3 Просмотр пайщика
- Просмотр детальной информации о пайщике
- Просмотр паевых взносов пайщика
- Формирование удостоверения пайщика
- Отображение в боковом меню

### 2.4 Удаление пайщика
- Удаление пайщика из системы
- Проверка наличия связанных данных
- Подтверждение удаления

### 2.5 Поиск и фильтрация
- Поиск пайщиков по ФИО
- Фильтрация по статусу
- Отображение результатов в таблице

### 2.6 Выбытие пайщика
- Процедура исключения пайщика
- Расчет суммы к возврату
- Формирование документов на возврат
- Изменение статуса пайщика

## 3. Модуль учета паевых взносов

### 3.1 Добавление паевого взноса
- Выбор пайщика
- Выбор типа взноса (вступительный, паевой, членский и др.)
- Выбор формы оплаты (наличные, безналичные, имущество)
- Ввод суммы или описания имущества
- Указание даты и описания

### 3.2 Редактирование паевого взноса
- Изменение данных взноса
- Обновление суммы или описания
- Изменение статуса оплаты
- Обновление связанных данных

### 3.3 Просмотр паевого взноса
- Просмотр детальной информации
- Отображение в боковом меню
- Просмотр связанной информации

### 3.4 Удаление паевого взноса
- Удаление паевого взноса
- Удаление связанных бухгалтерских проводок
- Подтверждение удаления

### 3.5 Учет имущественных взносов
- Ввод описания имущества
- Функция оценки имущества
- Обновление стоимости после оценки
- Отображение статуса оценки

### 3.6 Возврат паевых взносов
- Формирование документов на возврат
- Учет возврата в системе
- Создание бухгалтерских проводок возврата
