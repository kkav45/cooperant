# Пользовательская документация информационной системы учета пайщиков кооператива

## 1. Общие сведения

### 1.1 Назначение системы

Информационная система учета пайщиков кооператива предназначена для автоматизации процессов:
- Учета членства в кооперативе
- Контроля паевых взносов
- Ведения бухгалтерского учета
- Управления документооборотом
- Формирования отчетности

### 1.2 Кто может использовать систему

Система предназначена для:
- Сотрудников кооператива, ответственных за учет пайщиков
- Бухгалтеров, ведущих учет в кооперативе
- Руководства кооператива для получения отчетности
- Аудиторов для проверки деятельности кооператива

## 2. Начало работы

### 2.1 Запуск системы

1. Откройте файл `index.html` в браузере
2. Рекомендуется использовать Google Chrome, Microsoft Edge или Opera для полной функциональности
3. При первом запуске система загрузит демонстрационные данные (если они есть)

### 2.2 Основной интерфейс

При запуске вы увидите:
- Шапку с названием системы и навигацией
- Основное рабочее поле
- Главную страницу с общей статистикой

## 3. Работа с пайщиками

### 3.1 Просмотр списка пайщиков

1. Нажмите на пункт "Пайщики" в верхнем меню
2. В таблице будут отображены все зарегистрированные пайщики
3. Вы можете использовать поле поиска для фильтрации

### 3.2 Добавление нового пайщика

1. На странице "Пайщики" нажмите кнопку "Добавить пайщика"
2. Заполните обязательные поля:
   - ФИО (обязательно)
   - Статус (по умолчанию "Активный")
   - Дата вступления (по умолчанию сегодняшняя дата)
3. При необходимости укажите контактную информацию, адрес и примечания
4. Нажмите "Сохранить"

### 3.3 Редактирование данных пайщика

1. Найдите нужного пайщика в таблице
2. Нажмите кнопку "Изменить" в строке пайщика
3. Внесите необходимые изменения
4. Нажмите "Сохранить"

### 3.4 Удаление пайщика

1. Найдите пайщика в таблице
2. Нажмите кнопку "Удалить" в строке пайщика
3. Подтвердите удаление в появившемся диалоге

### 3.5 Поиск пайщика

1. На странице "Пайщики" используйте поле поиска в правой части
2. Введите часть ФИО или контактную информацию
3. Список автоматически отфильтруется

## 4. Работа с паевыми взносами

### 4.1 Просмотр паевых взносов

1. Нажмите на пункт "Паевые взносы" в меню
2. В таблице отображаются все зарегистрированные взносы
3. Для каждого взноса показаны: пайщик, тип, сумма, дата, статус

### 4.2 Добавление паевого взноса

1. На странице "Паевые взносы" нажмите кнопку "Добавить взнос"
2. Заполните форму:
   - Выберите пайщика из списка
   - Укажите тип взноса (вступительный, паевой, членский и др.)
   - Введите сумму
   - Укажите дату (по умолчанию сегодня)
   - При необходимости добавьте описание
   - Отметьте галочку "Оплачено", если взнос уже оплачен
3. Нажмите "Сохранить"

### 4.3 Редактирование паевого взноса

1. Найдите нужный взнос в таблице
2. Нажмите кнопку "Изменить"
3. Внесите необходимые изменения
4. Нажмите "Сохранить"

### 4.4 Отметка об оплате

1. Найдите неоплаченный взнос
2. Нажмите "Изменить"
3. Поставьте галочку "Оплачено"
4. Нажмите "Сохранить"

## 5. Бухгалтерский учет

### 5.1 Просмотр бухгалтерских проводок

1. Нажмите на пункт "Бухгалтерия" в меню
2. В таблице отображаются все бухгалтерские проводки
3. Для каждой проводки показаны: дата, дебет, кредит, сумма, описание

### 5.2 Добавление бухгалтерской проводки

1. На странице "Бухгалтерия" нажмите кнопку "Добавить проводку"
2. Заполните форму:
   - Укажите дату (по умолчанию сегодня)
   - Введите сумму
   - Укажите дебетовый счет (например, 51, 50, 76)
   - Укажите кредитовый счет (например, 86, 75, 90)
   - При необходимости добавьте описание
3. Нажмите "Сохранить"

### 5.3 Редактирование проводки

1. Найдите нужную проводку в таблице
2. Нажмите кнопку "Изменить"
3. Внесите необходимые изменения
4. Нажмите "Сохранить"

## 6. Документооборот

### 6.1 Просмотр документов

1. Нажмите на пункт "Документы" в меню
2. В таблице отображаются все зарегистрированные документы
3. Для каждого документа показаны: название, тип, дата, размер

### 6.2 Добавление документа

1. На странице "Документы" нажмите кнопку "Загрузить документ"
2. Заполните форму:
   - Выберите файл (поддерживаются PDF, DOC, XLS, JPG, PNG)
   - Укажите название документа
   - Выберите тип документа
   - При необходимости добавьте описание
3. Нажмите "Загрузить"

### 6.3 Просмотр информации о документе

1. Найдите нужный документ в таблице
2. Нажмите кнопку "Просмотр"
3. В модальном окне отобразится информация о документе

## 7. Формирование отчетов

### 7.1 Доступные отчеты

На странице "Отчеты" доступны следующие типы отчетов:
- Отчет по пайщикам
- Отчет по паевым взносам
- Бухгалтерский баланс
- Отчет о финансовых результатах

### 7.2 Генерация отчета

1. Нажмите на пункт "Отчеты" в меню
2. Выберите нужный тип отчета
3. Отчет отобразится в нижней части страницы
4. При необходимости вы можете сделать скриншот отчета

## 8. Сохранение и загрузка данных

### 8.1 Автоматическое сохранение

Система автоматически сохраняет данные при каждом изменении. При работе в браузере с поддержкой File System API:
- Данные сохраняются в файл `coop_data.json`
- Браузер запросит у вас место для сохранения файла

### 8.2 Резервное копирование

Для создания резервной копии:
1. Убедитесь, что все изменения сохранены
2. Найдите последний файл `coop_data.json`
3. Скопируйте его в безопасное место с уникальным именем

## 9. Решение проблем

### 9.1 Система не запускается

- Убедитесь, что вы открываете файл `index.html` в браузере
- Проверьте, что файлы `styles.css` и `app.js` находятся в той же директории
- Попробуйте очистить кэш браузера (Ctrl+F5)

### 9.2 Данные не сохраняются

- Проверьте, что вы используете поддерживаемый браузер
- Убедитесь, что у вас есть права на запись в директорию
- Проверьте настройки безопасности браузера

### 9.3 Не могу добавить пайщика/взнос

- Убедитесь, что все обязательные поля заполнены
- Проверьте, что вы не вводите недопустимые символы в числовые поля
- Попробуйте обновить страницу и повторить операцию

## 10. Рекомендации по использованию
