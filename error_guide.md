# Возможные ошибки и способы их устранения

## Ошибки при запуске приложения

### Ошибка: "File System API не поддерживается в вашем браузере"

**Причина:** Используемый браузер не поддерживает File System API.

**Решение:**
1. Обновите браузер до последней версии
2. Используйте один из поддерживаемых браузеров:
   - Google Chrome 86+
   - Microsoft Edge 86+
   - Opera 72+
3. Если невозможно использовать поддерживаемый браузер, приложение будет использовать localStorage с ограничениями по объему данных

### Ошибка: "Приложение не загружается"

**Причина:** Возможно, файлы приложения повреждены или неполностью скопированы.

**Решение:**
1. Проверьте целостность всех файлов приложения
2. Убедитесь, что файлы `index.html`, `styles.css`, и `app.js` находятся в одной директории
3. Откройте инструменты разработчика (F12) и проверьте наличие ошибок в консоли

## Ошибки при работе с данными

### Ошибка: "Данные не сохраняются"

**Причина:** 
- Браузер не поддерживает File System API
- Ограничения безопасности браузера
- Проблемы с доступом к файловой системе

**Решение:**
1. Проверьте, что вы используете поддерживаемый браузер
2. Убедитесь, что у вас есть права на запись в директорию с приложением
3. Попробуйте сохранить файл в другую директорию
4. Проверьте настройки безопасности браузера

### Ошибка: "Данные не загружаются"

**Причина:**
- Поврежденный файл данных
- Несовместимый формат файла
- Ошибка в структуре данных

**Решение:**
1. Проверьте, что файл данных в формате JSON
2. Проверьте структуру файла на соответствие описанной в `data_structure.md`
3. Используйте файл `demo_data.json` для проверки работоспособности
4. Попробуйте открыть файл в текстовом редакторе для проверки целостности

## Ошибки при работе с пайщиками

### Ошибка: "Не удается добавить нового пайщика"

**Причина:**
- Не заполнены обязательные поля
- Ошибка валидации данных
- Проблемы с сохранением данных

**Решение:**
1. Убедитесь, что поле "ФИО" заполнено
2. Проверьте, что все обязательные поля отмечены символом *
3. Попробуйте перезагрузить страницу и повторить операцию

### Ошибка: "Пайщик не отображается в списке"

**Причина:**
- Ошибка фильтрации
- Проблемы с отображением данных
- Данные не были сохранены

**Решение:**
1. Проверьте строку поиска - возможно, введены фильтры
2. Обновите страницу
3. Проверьте, что данные были успешно сохранены

## Ошибки при работе с паевыми взносами

### Ошибка: "Не удается добавить паевой взнос"

**Причина:**
- Не выбран пайщик
- Не указан тип взноса
- Не указана сумма
- Пайщик удален из системы

**Решение:**
1. Убедитесь, что выбран пайщик из списка
2. Проверьте, что указан тип взноса
3. Убедитесь, что сумма указана числом
4. Проверьте, что пайщик все еще существует в системе

### Ошибка: "Сумма отображается некорректно"

**Причина:**
- Неправильный формат ввода суммы
- Ошибки округления

**Решение:**
1. Вводите сумму только цифрами
2. Используйте точку как десятичный разделитель
3. Проверьте, что сумма не превышает допустимый диапазон

## Ошибки при работе с бухгалтерскими проводками

### Ошибка: "Не удается создать бухгалтерскую проводку"

**Причина:**
- Не заполнены обязательные поля
- Некорректный формат счета
- Не указана сумма

**Решение:**
1. Убедитесь, что все обязательные поля заполнены
2. Проверьте формат номера счета (обычно 2-4 цифры)
3. Убедитесь, что сумма указана числом

## Ошибки при работе с документами

### Ошибка: "Не удается загрузить документ"

**Причина:**
- Неподдерживаемый формат файла
- Файл слишком велик
- Не указано название документа

**Решение:**
1. Убедитесь, что файл имеет поддерживаемый формат (.pdf, .doc, .docx, .xls, .xlsx, .jpg, .jpeg, .png)
2. Проверьте, что название документа указано
3. В текущей реализации файлы не сохраняются физически, только метаданные

### Ошибка: "Документ не отображается"

**Причина:**
- В текущей реализации файлы не сохраняются физически

**Решение:**
1. Это ограничение текущей реализации
2. В реальном приложении файлы будут сохраняться с использованием File System API

## Ошибки в отчетах

### Ошибка: "Отчет не генерируется"

**Причина:**
- Нет данных для формирования отчета
- Ошибка в логике генерации отчета

**Решение:**
1. Убедитесь, что в системе есть данные
2. Проверьте, что отчет соответствует выбранному типу
3. Обновите страницу и повторите попытку

## Ошибки в модальных окнах

### Ошибка: "Модальное окно не закрывается"

**Причина:**
- Ошибка в JavaScript
- Проблемы с отображением

**Решение:**
1. Нажмите клавишу Escape
2. Кликните вне модального окна
3. Обновите страницу, если проблема сохраняется

## Общие рекомендации по устранению ошибок

1. **Проверьте консоль браузера**
   - Откройте инструменты разработчика (F12)
   - Перейдите на вкладку Console
   - Проверьте наличие ошибок JavaScript

2. **Очистите кэш браузера**
   - Иногда старые файлы могут мешать работе
   - Используйте Ctrl+F5 для принудительной перезагрузки

3. **Проверьте права доступа**
   - Убедитесь, что у вас есть права на чтение и запись файлов приложения

4. **Используйте поддерживаемые браузеры**
   - Приложение оптимизировано для Chrome, Edge и Opera
   - В других браузерах могут возникать проблемы с File System API

5. **Создайте резервную копию**
   - Перед выполнением любых действий по устранению ошибок создайте резервную копию данных

## Контакты для поддержки

Если вы не можете устранить ошибку самостоятельно:
- Проверьте файлы в директории приложения
- Убедитесь, что все файлы соответствуют описанию в документации
- При необходимости свяжитесь с разработчиком для получения дополнительной помощи
