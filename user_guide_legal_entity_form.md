# Руководство по использованию формы юридического лица

## Общая информация

Форма добавления юридического лица позволяет внести информацию о юридическом лице в систему учета пайщиков кооператива.

## Как использовать

1. Откройте форму добавления юридического лица (кнопка "Добавить юр.лицо")
2. Заполните все обязательные поля:
   - Наименование организации
   - Место нахождения (юридический адрес)
   - ОГРН
   - ИНН
   - КПП (опционально)
   - Банковские реквизиты (наименование банка, БИК, корреспондентский и расчетный счет, назначение платежа)
   - Данные представителя (ФИО, должность, контактный телефон, email, документ, удостоверяющий личность)
   - Информация о доверенности (номер, дата выдачи, кем выдана)
3. Нажмите кнопку "Сохранить"

## Поля формы

### Данные юридического лица
- Наименование организации (обязательно)
- Место нахождения (юридический адрес) (обязательно)
- ОГРН (обязательно)
- ИНН (обязательно)
- КПП (опционально)

### Банковские реквизиты
- Наименование банка (обязательно)
- БИК банка (обязательно)
- Корреспондентский счёт (обязательно)
- Расчётный счёт (обязательно)
- Назначение платежа (обязательно)

### Данные представителя юридического лица
- ФИО (обязательно)
- Должность (обязательно)
- Контактный телефон (обязательно)
- Email (обязательно)
- Документ, удостоверяющий личность (обязательно)

### Доверенность
- Номер доверенности (обязательно)
- Дата выдачи (обязательно)
- Кем выдана (обязательно)

## Ограничения

- Все обязательные поля должны быть заполнены для сохранения формы
- Форма не содержит раздела "Паевой взнос"
- Форма не содержит автоматического заполнения по ИНН
