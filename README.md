# Информационная система учета пайщиков кооператива

## Описание

Это веб-приложение для учета пайщиков, паевых взносов и бухгалтерского учета в потребительской кооперации. Приложение работает локально в браузере и использует File System API для сохранения данных и хранения документов в указанной папке. Система поддерживает внесение паевых взносов не только деньгами, но и иным имуществом, а также процедуру выбытия пайщиков с возвратом паевых взносов.

## Функциональные возможности

- Учет пайщиков (добавление, редактирование, удаление, поиск)
- Учет паевых взносов (вступительные, паевые, членские и др.) в денежной и натуральной форме
- Механизм оценки имущества, вносимого в качестве паевого взноса
- Процедура выбытия пайщиков с возвратом паевых взносов
- Бухгалтерский учет (автоматическое проведение бухгалтерских проводок при операциях с паевыми взносами)
- Документооборот (загрузка и хранение документов в папке C:\КООПЕРАНТ)
- Формирование отчетов

## Требования

- Современный браузер с поддержкой File System API (Chrome 86+, Edge 86+, Opera 72+)
- Для браузеров без поддержки File System API используется localStorage (ограничения по объему данных)

## Установка и запуск

1. Создайте папку C:\КООПЕРАНТ на вашем компьютере
2. Откройте файл `index.html` в браузере
3. При первом использовании раздела "Документы" настройте хранение, выбрав папку C:\КООПЕРАНТ
4. Приложение готово к использованию

## Использование

### Учет пайщиков
1. Перейдите в раздел "Пайщики"
2. Используйте кнопку "Добавить пайщика" для добавления нового члена кооператива
3. Для редактирования или удаления пайщика используйте соответствующие кнопки в таблице
4. Для выбытия пайщика используйте кнопку "Выбытие" (доступна только для активных пайщиков)

### Учет паевых взносов
1. Перейдите в раздел "Паевые взносы"
2. Используйте кнопку "Добавить взнос" для регистрации нового взноса
3. Выберите пайщика, тип взноса
4. Выберите форму оплаты: наличные, безналичные или имущество
5. Для имущественных взносов обязательно проведите оценку (кнопка "Оценить" появляется для неоцененных взносов)

### Выбытие пайщика
1. В разделе "Пайщики" найдите активного пайщика
2. Нажмите кнопку "Выбытие" в строке пайщика
3. Укажите причину и дату выбытия
4. В открывшейся форме возврата укажите сумму и форму возврата паевого взноса
5. Подтвердите операцию

### Бухгалтерский учет
1. Перейдите в раздел "Бухгалтерия"
2. Используйте кнопку "Добавить проводку" для ввода бухгалтерской проводки
3. Укажите дебетовый и кредитовый счета, сумму и описание
4. При внесении и возврате паевых взносов проводки создаются автоматически

### Документооборот
1. Перейдите в раздел "Документы"
2. Нажмите "Настроить хранение" и выберите папку C:\КООПЕРАНТ
3. Используйте кнопку "Загрузить документ" для добавления нового документа
4. Выберите файл, укажите название и тип документа
5. Документ будет сохранен в соответствующей подпапке (Contracts, Reports, Payments, Other)

### Заявления на вступление
1. Перейдите в раздел "Заявления"
2. Нажмите "Подать заявление" для открытия формы заявления
3. Заполните все обязательные поля
4. Подтвердите согласие с условиями
5. Заявление будет автоматически сохранено в подпапке Applications папки C:\КООПЕРАНТ

### Заседания и протоколы
1. Перейдите в раздел "Заседания"
2. Нажмите "Назначить заседание" для планирования встречи
3. Заполните информацию о заседании
4. После проведения заседания сформируйте протокол
5. Протокол будет автоматически сохранен в подпапке Protocols папки C:\КООПЕРАНТ

### Удостоверения пайщиков
1. При принятии нового пайщика автоматически формируется удостоверение
2. Удостоверение сохраняется в подпапке Certificates папки C:\КООПЕРАНТ
3. Также доступна функция просмотра и печати удостоверения

### Отчеты
1. Перейдите в раздел "Отчеты"
2. Выберите нужный тип отчета для генерации
3. В отчетах по паевым взносам теперь отображаются как обычные взносы, так и возвраты

## Новые возможности


### Внесение паевых взносов имуществом
- При добавлении взноса можно выбрать форму оплаты "Имущество"
- После добавления необходимо провести оценку имущества
- Для неоцененных имущественных взносов отображается статус "Не оценено"

### Оценка имущества
- В таблице паевых взносов для неоцененных имущественных взносов появляется кнопка "Оценить"
- При оценке указывается стоимость, метод оценки, оценщик и примечания
- После оценки сумма взноса обновляется

### Выбытие пайщиков
- Возможность исключения пайщика из кооператива
- Автоматический расчет суммы к возврату
- Возможность возврата взноса в денежной или натуральной форме
- Проверка наличия задолженности перед выбытием

### Автоматические бухгалтерские проводки
- При внесении паевого взноса автоматически создается проводка
- При возврате паевого взноса автоматически создается проводка возврата
- Все проводки связаны с соответствующими операциями

## Сохранение и загрузка данных

Для сохранения данных используйте File System API, который доступен в поддерживающих браузерах:
- Нажмите кнопку сохранения (в текущей реализации автоматически при каждом изменении)
- Выберите место для сохранения файла данных

Для загрузки ранее сохраненных данных:
- При следующем открытии приложения данные автоматически загрузятся из файла

## Совместимость

Приложение использует File System API, который поддерживается не во всех браузерах:
- Поддерживается: Chrome 86+, Edge 86+, Opera 72+
- Ограниченная поддержка: другие Chromium-браузеры
- Не поддерживается: Firefox, Safari (на момент создания)

Для браузеров без поддержки File System API реализован резервный вариант с использованием localStorage, но с ограничениями по объему данных.

## Безопасность

Все данные хранятся локально на вашем компьютере. Приложение не отправляет данные на серверы.
Документы хранятся в папке C:\КООПЕРАНТ, доступ к которой контролируется пользователем.
