# Инструкция по использованию обновленной системы с File System API

## 1. Подготовка к работе

### 1.1 Создание папки C:\КООПЕРАНТ
1. Откройте Проводник Windows
2. Убедитесь, что вы видите диск C:\
3. Щелкните правой кнопкой мыши по свободному месту
4. Выберите "Создать" → "Папка"
5. Назовите папку "КООПЕРАНТ"
6. Убедитесь, что папка создана по пути: C:\КООПЕРАНТ

### 1.2 Проверка браузера
Для корректной работы системы с File System API рекомендуется использовать:
- Google Chrome версии 86 или выше
- Microsoft Edge версии 86 или выше
- Opera версии 72 или выше

## 2. Запуск системы

### 2.1 Открытие системы
1. Откройте выбранный браузер
2. Перейдите в папку с системой
3. Дважды щелкните по файлу index.html
4. Система откроется в новой вкладке браузера

## 3. Настройка хранения документов

### 3.1 Первичная настройка
1. В системе перейдите на вкладку "Документы"
2. Нажмите кнопку "Настроить хранение"
3. В появившемся окне найдите и выберите папку C:\КООПЕРАНТ
4. Нажмите "Разрешить доступ" или "Выбрать папку"
5. Система создаст необходимые подпапки: Contracts, Reports, Payments, Other

### 3.2 Подтверждение настройки
После успешной настройки вы увидите сообщение: "Директория для хранения документов успешно настроена!"

## 4. Работа с документами

### 4.1 Загрузка документа
1. Убедитесь, что настройка хранения выполнена
2. Нажмите кнопку "Загрузить документ"
3. В открывшейся форме:
   - Выберите файл с помощью кнопки "Выбрать файл"
   - Введите название документа
   - Выберите тип документа (Договор, Отчет, Платежный документ, Прочее)
   - При необходимости добавьте описание
4. Нажмите "Загрузить"
5. Документ будет сохранен в соответствующей подпапке в C:\КООПЕРАНТ

### 4.2 Просмотр документа
1. Найдите нужный документ в таблице
2. Нажмите кнопку "Просмотр" в строке документа
3. В открывшемся окне вы увидите:
   - Тип документа
   - Дату
   - Размер
   - Описание
   - Имя файла
4. Используйте кнопки:
   - "Открыть файл" - для открытия документа во внешней программе
   - "Скачать" - для загрузки копии документа
   - "Закрыть" - для закрытия окна просмотра

### 4.3 Удаление документа
1. Найдите документ в таблице
2. Нажмите кнопку "Удалить" в строке документа
3. Подтвердите удаление в появившемся диалоге
4. Документ будет удален как из системы, так и из папки C:\КООПЕРАНТ

## 5. Структура папок

После настройки система создаст следующую структуру в папке C:\КООПЕРАНТ:

```
C:\КООПЕРАНТ\
├── Contracts\     # Договоры
├── Reports\       # Отчеты
├── Payments\      # Платежные документы
└── Other\         # Прочие документы
```

Документы автоматически распределяются по соответствующим подпапкам в зависимости от их типа.

## 6. Резервное копирование

### 6.1 Резервное копирование документов
Для создания резервной копии документов:
1. Закройте систему
2. Скопируйте всю папку C:\КООПЕРАНТ
3. Сохраните копию в безопасном месте

### 6.2 Восстановление из резервной копии
Для восстановления документов:
1. Закройте систему
2. Удалите содержимое папки C:\КООПЕРАНТ
3. Скопируйте содержимое резервной копии в C:\КООПЕРАНТ

## 7. Возможные проблемы и решения

### 7.1 Ошибка доступа к папке
**Проблема:** "Не удалось получить доступ к директории"
**Решение:** 
- Убедитесь, что вы выбрали именно папку C:\КООПЕРАНТ
- Проверьте, что у вас есть права на запись в эту папку
- Попробуйте запустить браузер от имени администратора

### 7.2 Файл не открывается
**Проблема:** "Не удалось открыть файл"
**Решение:**
- Убедитесь, что на вашем компьютере установлены программы для открытия данного типа файлов
- Проверьте, не заблокирован ли файл антивирусом

### 7.3 Файл не скачивается
**Проблема:** "Не удалось скачать файл"
**Решение:**
- Проверьте настройки браузера, возможно, загрузки заблокированы
- Убедитесь, что настройка хранения документов выполнена

## 8. Безопасность

### 8.1 Защита документов
- Папка C:\КООПЕРАНТ содержит важные документы
- Обеспечьте физическую безопасность компьютера
- Регулярно создавайте резервные копии
- Используйте антивирусное программное обеспечение

### 8.2 Ограничение доступа
- Ограничьте доступ к папке C:\КООПЕРАНТ посторонним пользователям
- Используйте пароли для входа в систему
- Регулярно обновляйте программное обеспечение

## 9. Техническая поддержка

При возникновении проблем:
1. Проверьте версию браузера
2. Убедитесь, что выполнены все шаги настройки
3. Обратитесь к файлу error_guide.md для решения типичных проблем
4. При необходимости свяжитесь с технической поддержкой
