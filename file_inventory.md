# Перечень файлов информационной системы учета пайщиков кооператива

## 1. Основные файлы системы

### 1.1 index.html
- **Тип:** HTML-документ
- **Назначение:** Главная страница веб-приложения
- **Описание:** Структура пользовательского интерфейса, навигация, основные элементы управления
- **Размер:** ~15 КБ
- **Версия:** 1.0

### 1.2 app.js
- **Тип:** JavaScript-файл
- **Назначение:** Основная логика приложения
- **Описание:** Функции для работы с пайщиками, паевыми взносами, бухгалтерским учетом, документооборотом, заявлениями, заседаниями
- **Размер:** ~3.5 МБ
- **Версия:** 1.0

### 1.3 styles.css
- **Тип:** CSS-файл
- **Назначение:** Стили оформления интерфейса
- **Описание:** Визуальное оформление элементов интерфейса, адаптивный дизайн, стили для модальных окон и бокового меню
- **Размер:** ~4 КБ
- **Версия:** 1.0
