# Каталог таблиц и форм информационной системы учета пайщиков кооператива

## 1. Общая информация

### 1.1 Назначение каталога
Каталог содержит описание всех таблиц и форм, используемых в информационной системе учета пайщиков кооператива, включая их структуру, поля и функциональность.

### 1.2 Классификация элементов
- **Таблицы:** Элементы для отображения списков данных
- **Формы:** Элементы для ввода и редактирования данных
- **Модальные окна:** Временные интерфейсы для специфических операций
- **Боковые меню:** Интерфейсы для просмотра детальной информации (40% ширины экрана)

## 2. Таблицы системы

### 2.1 Таблица пайщиков (members-table)

#### 2.1.1 Описание
Таблица для отображения списка всех пайщиков кооператива с возможностью поиска, фильтрации и управления.

#### 2.1.2 Структура
| Колонка | Название | Тип данных | Описание |
|---------|----------|------------|----------|
| 1 | ID | Строка | Уникальный идентификатор пайщика |
| 2 | ФИО | Строка | Полное имя пайщика |
| 3 | Дата вступления | Дата | Дата вступления в кооператив |
| 4 | Статус | Строка | Текущий статус пайщика |
| 5 | Контакт | Строка | Контактная информация |
| 6 | Действия | Кнопки | Кнопки управления (просмотр, изменить, выбытие, удалить) |

#### 2.1.3 Функциональность
- Поиск по ФИО и контактам
- Фильтрация по статусу
- Сортировка по колонкам
- Пагинация (в будущем)
- Действия: Просмотр, Редактирование, Выбытие, Удаление

#### 2.1.4 Источник данных
- Массив `members` в app.js
- Обновляется при изменении данных
- Отображается в реальном времени

### 2.2 Таблица паевых взносов (payments-table)

#### 2.2.1 Описание
Таблица для отображения всех паевых взносов, включая денежные и имущественные.

#### 2.2.2 Структура
| Колонка | Название | Тип данных | Описание |

## 3. Формы системы

### 3.1 Форма добавления пайщика

#### 3.1.1 Описание
Форма для добавления нового пайщика в систему с вводом всех необходимых данных.

#### 3.1.2 Поля формы
| № | Название поля | Тип элемента | Обязательное | Описание |
|---|---------------|--------------|--------------|----------|
| 1 | ФИО | Текстовое поле | Да | Полное имя пайщика |
| 2 | Статус | Выпадающий список | Нет | Статус пайщика (активный, кандидат и др.) |
| 3 | Дата вступления | Поле даты | Нет | Дата вступления в кооператив |
| 4 | Контакт | Текстовое поле | Нет | Контактная информация |
| 5 | Адрес | Текстовое поле (многострочное) | Нет | Адрес проживания |
| 6 | Примечания | Текстовое поле (многострочное) | Нет | Дополнительная информация |

#### 3.1.3 Валидация
- Проверка обязательного поля ФИО
- Проверка формата даты
- Проверка корректности контактной информации

#### 3.1.4 Действия
- Сохранить: Добавление пайщика в систему
- Отмена: Закрытие формы без сохранения

### 3.2 Форма редактирования пайщика

#### 3.2.3 Валидация
- Проверка корректности изменений
- Проверка формата даты
- Проверка соответствия бизнес-правилам

#### 3.2.4 Действия

- Отмена: Закрытие формы без изменений

### 3.3 Форма добавления паевого взноса

#### 3.3.1 Описание
Форма для добавления нового паевого взноса от пайщика.

#### 3.3.2 Поля формы
| № | Название поля | Тип элемента | Обязательное | Описание |
|---|---------------|--------------|--------------|----------|
| 1 | Пайщик | Выпадающий список | Да | Выбор из существующих пайщиков |
| 2 | Тип взноса | Выпадающий список | Да | Вступительный, паевой, членский и др. |
| 3 | Форма оплаты | Выпадающий список | Да | Наличные, безналичные, имущество |
| 4 | Сумма | Числовое поле | Да (если не имущество) | Размер взноса |
| 5 | Описание имущества | Текстовое поле (многострочное) | Да (если форма "имущество") | Описание передаваемого имущества |
| 6 | Дата | Поле даты | Да | Дата внесения взноса |
| 7 | Описание | Текстовое поле (многострочное) | Нет | Дополнительное описание |
| 8 | Статус оплаты | Чекбокс | Нет | Отметка об оплате |
| 9 | Номер документа | Текстовое поле | Нет | Номер квитанции или акта |

#### 3.3.3 Условная логика
- При выборе формы "имущество" скрывается поле "Сумма", появляется "Описание имущества"
- При других формах наоборот

#### 3.3.4 Валидация
- Проверка обязательных полей
- Проверка корректности суммы
- Проверка формата даты

#### 3.3.5 Действия
- Сохранить: Добавление взноса в систему
- Отмена: Закрытие формы без сохранения
