## 2. Модуль отчетности (детальная проработка)

### 2.1. Три типа учета в левой панели

#### **📒 Бухгалтерский учет**
- Оборотно-сальдовая ведомость
- Анализ счета
- Карточка счета
- Журнал-ордер
- Главная книга

#### **📊 Налоговый учет**
- Налоговая декларация (УСН)
- Книга доходов и расходов (КУДиР)
- Расчет страховых взносов
- 6-НДФЛ
- 2-НДФЛ

#### **📈 Управленческий учет**
- P&L (Отчет о прибылях и убытках)
- Cash Flow (Движение денежных средств)
- Баланс (управленческий)
- Анализ дебиторской задолженности
- Анализ кредиторской задолженности
- Отчет по пайщикам (детальный)

## 7. Сценарии использования

### Сценарий 1: Просмотр управленческого отчета

```
1. Пользователь в левой панели нажимает "📈 Управленческий учет"
   ↓
2. Открывается полноэкранное меню с карточками отчетов
   ↓
3. Пользователь выбирает "P&L (Прибыли и убытки)"
   ↓
4. Справа выезжает панель с отчетом P&L
   ↓
5. Пользователь видит:
   - Таблицу с доходами/расходами
   - График динамики прибыли
   - Сравнение с предыдущим периодом
   - Быстрые действия (аналитика, экспорт)
   ↓
6. Пользователь может:
   - Переключиться на график (вместо таблицы)
   - Изменить период
   - Экспортировать в PDF/Excel
   - Отправить по email
```

### Сценарий 2: Сверка бухгалтерских данных

```
1. Пользователь выбирает "📒 Бухгалтерский учет"
   ↓
2. Выбирает "Оборотно-сальдовая ведомость"
   ↓
3. В правой панели открывается ОСВ
   ↓
4. Пользователь кликает на строку со счетом
   ↓
5. Открывается детализация по счету (анализ счета)
   ↓
6. Пользователь видит все проводки по счету
   ↓
7. Может экспортировать или распечатать
```

### Сценарий 3: Подготовка налоговой декларации

```
1. Пользователь выбирает "📊 Налоговый учет"
   ↓
2. Выбирает "Декларация по УСН"
   ↓
3. В правой панели открывается форма декларации
   ↓
4. Данные автоматически заполняются из системы
   ↓
5. Пользователь проверяет разделы:
   - Раздел 1 (сумма налога)
   - Раздел 2 (расчет налоговой базы)
   ↓
6. Может:
   - Сформировать файл для ФНС (XML)
   - Распечатать для подписи
   - Отправить через интеграцию
```
