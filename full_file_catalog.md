## 7. Версии и обновления

Все файлы находятся в версии 1.0, за исключением:
- app.js: основной файл системы, постоянно обновляется
- index.html: может обновляться при добавлении новых функций
- styles.css: может обновляться при изменении интерфейса

## 8. Зависимости

- app.js зависит от index.html и styles.css
- Все документы могут ссылаться на техническую спецификацию
- Система требует поддержки File System API в браузере
