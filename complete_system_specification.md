# Полное техническое описание информационной системы учета пайщиков кооператива

## 1. Общая характеристика системы

### 1.1 Назначение
Информационная система учета пайщиков кооператива предназначена для автоматизации процессов управления потребительской кооперацией, включая учет пайщиков, паевых взносов, бухгалтерский учет, документооборот и формирование отчетности.

### 1.2 Основные функции
- Учет пайщиков и управление их статусами
- Учет паевых взносов (в денежной и натуральной форме)
- Бухгалтерский учет и формирование проводок
- Документооборот и хранение документов
- Обработка заявлений на вступление
- Проведение заседаний и формирование протоколов
- Формирование удостоверений пайщиков
- Создание отчетов и аналитики

### 1.3 Соответствие требованиям
- Закон РФ от 19.06.1992 N 3085-1 "О потребительской кооперации"
- Требования бухгалтерского учета
- Требования по защите персональных данных
- Современные веб-стандарты

## 5. Структура данных

### 5.1 Пайщики
```javascript
{
  id: string,                    // Уникальный идентификатор
  name: string,                  // ФИО пайщика
  status: enum,                  // Статус (active, candidate, suspended, excluded, withdrawn)
  joinDate: string,              // Дата вступления
  contact: string,               // Контактная информация
  address: string,               // Адрес
  notes: string,                 // Примечания
  createdAt: string,             // Дата создания
  updatedAt: string              // Дата обновления
}
```

### 5.2 Паевые взносы
```javascript
{
  id: string,                    // Уникальный идентификатор
  memberId: string,              // Связь с пайщиком
  type: enum,                    // Тип взноса (entrance, share, membership и др.)
  method: enum,                  // Метод оплаты (cash, non_cash, property)
  amount: number,                // Сумма (для имущества - после оценки)
  propertyDescription: string,   // Описание имущества (если method='property')
  date: string,                  // Дата взноса
  description: string,           // Описание
  paid: boolean,                 // Статус оплаты
  documentNumber: string,        // Номер документа
  expected: boolean,             // Признак ожидаемого взноса
  applicationId: string,         // Связь с заявлением (если есть)
  createdAt: string,             // Дата создания
  updatedAt: string              // Дата обновления
}
```

### 5.3 Бухгалтерские проводки
```javascript
{
  id: string,                    // Уникальный идентификатор
  date: string,                  // Дата проводки
  amount: number,                // Сумма
  debitAccount: string,          // Дебетовый счет
  creditAccount: string,         // Кредитовый счет
  description: string,           // Описание
  relatedPaymentId: string,      // Связь с паевым взносом
  transactionType: enum,         // Тип операции (incoming, return)
  createdAt: string,             // Дата создания
  updatedAt: string              // Дата обновления
}
```

### 5.4 Заявления на вступление
```javascript
{
  id: string,                    // Уникальный идентификатор
  applicantName: string,         // ФИО заявителя
  applicantContact: string,      // Контактная информация
  birthDate: string,             // Дата рождения
  passport: string,              // Паспортные данные
  registrationAddress: string,   // Адрес регистрации
  residenceAddress: string,      // Адрес проживания
  occupation: string,            // Род занятий
  income: string,                // Доход
  desiredShareAmount: number,    // Желаемый размер взноса
  paymentMethod: enum,           // Метод оплаты
  propertyDescription: string,   // Описание имущества (если paymentMethod='property')
  additionalInfo: string,        // Дополнительная информация
  submissionDate: string,        // Дата подачи
  status: enum,                  // Статус (pending, approved, rejected)
  decisionNotes: string,         // Комментарии к решению
  processedAt: string,           // Дата обработки
  createdAt: string              // Дата создания
}
```

## 7. Процессы системы

5. Формирование отчетов

### 7.3 Процесс проведения заседаний
1. Назначение заседания
2. Проведение заседания
3. Фиксация участников
4. Фиксация принятых решений

### 7.4 Процесс формирования отчетности
