# Инструкция по запуску и использованию системы

## 1. Подготовка к работе

### 1.1 Создание папки C:\КООПЕРАНТ
1. Нажмите клавиши Win+R на клавиатуре
2. Введите `cmd` и нажмите Enter
3. В командной строке введите:
   ```
   mkdir C:\КООПЕРАНТ
   ```
4. Нажмите Enter для создания папки

### 1.2 Проверка браузера
1. Убедитесь, что у вас установлен один из следующих браузеров:
   - Google Chrome версии 86 или выше
   - Microsoft Edge версии 86 или выше
   - Opera версии 72 или выше
2. Обновите браузер до последней версии для лучшей совместимости

## 2. Запуск системы

### 2.1 Первый запуск
1. Откройте папку с системой (d:\!Минкооп_рф\! КООП\)
2. Дважды щелкните по файлу `index.html`
3. Система откроется в браузере
4. При первом запуске появится модальное окно с инструкцией по настройке папки C:\КООПЕРАНТ
5. Нажмите кнопку "Настроить доступ к папке"
6. В открывшемся окне выберите папку C:\КООПЕРАНТ
7. Подтвердите доступ к папке в диалоговом окне браузера

### 2.2 Повторный запуск
1. Откройте файл `index.html` в поддерживаемом браузере
2. Система загрузится с последними сохраненными данными

## 3. Основные разделы системы

### 3.1 Главная страница (Dashboard)
- Отображает общую статистику системы
- Показывает количество пайщиков, взносов, задолженность
- Предоставляет быстрый доступ к основным функциям

### 3.2 Пайщики
- **Добавление пайщика:** Нажмите "Добавить пайщика", заполните форму, нажмите "Сохранить"
- **Просмотр пайщика:** Нажмите "Просмотр" в строке пайщика (информация отображается в боковом меню)
- **Редактирование:** Нажмите "Изменить" в строке пайщика
- **Выбытие:** Нажмите "Выбытие" для исключения пайщика с возвратом паевого взноса
- **Удаление:** Нажмите "Удалить" (только для кандидатов)

### 3.3 Паевые взносы
- **Добавление взноса:** Нажмите "Добавить взнос", заполните форму, нажмите "Сохранить"
- **Форма оплаты:** Выберите "Наличные", "Безналичные" или "Имущество"
- **Для имущества:** Введите описание имущества, сумма будет установлена при оценке
- **Оценка имущества:** Нажмите "Оценить" для неоцененного имущественного взноса
- **Просмотр:** Нажмите "Просмотр" для детальной информации (в боковом меню)

### 3.4 Бухгалтерия
- **Добавление проводки:** Нажмите "Добавить проводку", заполните данные, нажмите "Сохранить"
- **Просмотр проводки:** Нажмите "Просмотр" для детальной информации (в боковом меню)
- **Редактирование:** Нажмите "Изменить" в строке проводки
- **Удаление:** Нажмите "Удалить" в строке проводки

### 3.5 Документы
- **Загрузка документа:** Нажмите "Загрузить документ", выберите файл, укажите название и тип
- **Просмотр документа:** Нажмите "Просмотр" для информации о документе (в боковом меню)
- **Открытие файла:** Нажмите "Открыть файл" для открытия документа во внешней программе
- **Скачивание:** Нажмите "Скачать" для сохранения копии документа

### 3.6 Заявления
- **Подача заявления:** Нажмите "Подать заявление", заполните форму в отдельном окне
- **Просмотр заявления:** Нажмите "Просмотр" для детальной информации (в боковом меню)
- **Обработка заявления:** Нажмите "Обработать" для принятия решения
- **Пакетное рассмотрение:** Нажмите "Пакетное рассмотрение" для обработки нескольких заявлений

### 3.7 Заседания
- **Назначение заседания:** Нажмите "Назначить заседание", заполните данные
- **Проведение заседания:** Нажмите "Провести" для заполнения протокола
- **Просмотр протокола:** Нажмите "Протокол" для просмотра (в модальном окне)

### 3.8 Отчеты
- **Генерация отчета:** Выберите тип отчета для формирования
- **Доступные отчеты:**
  - Отчет по пайщикам
  - Отчет по паевым взносам
  - Бухгалтерский баланс
  - Отчет о финансовых результатах

## 4. Работа с боковым меню (40% ширины экрана)

### 4.1 Просмотр детальной информации
- При нажатии кнопки "Просмотр" информация отображается в боковом меню
- Боковое меню занимает 40% ширины экрана и перекрывает основной интерфейс
- Для закрытия бокового меню нажмите крестик в заголовке или кнопку "Закрыть"

### 4.2 Действия из бокового меню
- Из бокового меню доступны действия по редактированию и удалению
- Некоторые действия могут быть недоступны в зависимости от статуса записи

## 5. Работа с File System API

### 5.1 Сохранение данных
- Данные автоматически сохраняются при каждом изменении
- Файл данных сохраняется в папке C:\КООПЕРАНТ\Data\
- Имя файла: coop_data_YYYY-MM-DD_HHMMSS.json (где YYYY-MM-DD - дата, HHMMSS - время)

### 5.2 Резервное копирование
- Система автоматически создает резервные копии при изменениях
- Резервные копии хранятся в папке C:\КООПЕРАНТ\Backups\
- Для ручного создания резервной копии используйте соответствующую функцию в системе

### 5.3 Восстановление данных
- Для восстановления данных из резервной копии используйте функцию загрузки данных
- Выберите файл резервной копии для восстановления

## 6. Безопасность данных

### 6.1 Защита информации
- Все данные хранятся локально на вашем компьютере
- Никакие данные не передаются на серверы
- Используется File System API с согласия пользователя
- Регулярно создавайте резервные копии данных

### 6.2 Резервное копирование
- Создавайте резервные копии ежедневно
- Храните копии в безопасном месте
- Проверяйте возможность восстановления из копий

## 7. Решение проблем

### 7.1 Система не запускается
- Убедитесь, что используете поддерживаемый браузер
- Проверьте, что файлы index.html, app.js и styles.css находятся в одной папке
- Попробуйте очистить кэш браузера (Ctrl+F5)

### 7.2 Данные не сохраняются
- Проверьте, что вы предоставили доступ к папке C:\КООПЕРАНТ
- Убедитесь, что у вас есть права на запись в эту папку
- Проверьте настройки безопасности браузера

### 7.3 Ошибка File System API
- Убедитесь, что используете поддерживаемый браузер
- Проверьте, что включена поддержка File System API
- Попробуйте перезапустить браузер

### 7.4 Проблемы с интерфейсом
- Проверьте, что все CSS и JS файлы загрузились корректно
- Обновите страницу (Ctrl+F5)
- Проверьте консоль браузера (F12) на наличие ошибок
