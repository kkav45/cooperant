## БЫСТРЫЙ СТАРТ

### Для пайщика:

```
1. Откройте index.html
2. Выберите «👤 Пайщик»
3. Нажмите «Продолжить»
4. Авторизуйтесь через Яндекс
5. Готово!
```

**Тестирование:**

- Вкладка «Поиск» → найдите кооператив
- Нажмите «Подать заявление» → заполните форму
- Вкладка «Мои заявления» → проверьте статус

### Для кооператива:

```
1. Откройте index.html
2. Выберите «🏢 Кооператив»
3. Нажмите «Продолжить»
4. Готово!
```

**Доступные модули:**

- Панель управления (демо-статистика)
- Модули в разработке (следите за обновлениями)
