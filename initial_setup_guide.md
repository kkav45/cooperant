# Инструкция по первоначальной настройке системы

## 1. Подготовка системы

### 1.1 Создание папки для хранения документов
1. Нажмите клавиши Win + R на клавиатуре
2. Введите `explorer.exe` и нажмите Enter
3. В открывшемся окне Проводника перейдите на диск C:\
4. Щелкните правой кнопкой мыши по свободному месту
5. Выберите "Создать" → "Папка"
6. Назовите папку "КООПЕРАНТ"
7. Убедитесь, что папка создана по пути: C:\КООПЕРАНТ

### 1.2 Проверка браузера
1. Откройте ваш браузер (рекомендуется Chrome, Edge или Opera последних версий)
2. Перейдите в меню "Помощь" или "О программе"
3. Убедитесь, что версия браузера:
   - Chrome 86 или выше
   - Edge 86 или выше
   - Opera 72 или выше

## 2. Запуск системы

### 2.1 Открытие системы
1. Перейдите в папку, где находятся файлы системы
2. Найдите файл `index.html`
3. Дважды щелкните по файлу для открытия в браузере
4. Система загрузится в новой вкладке браузера

## 3. Настройка хранения документов

### 3.1 Первый запуск раздела "Документы"
1. В интерфейсе системы нажмите на вкладку "Документы" в верхнем меню
2. Вы увидите сообщение о необходимости настройки хранения
3. Нажмите кнопку "Настроить хранение"

### 3.2 Выбор папки C:\КООПЕРАНТ
1. В появившемся окне навигации:
   - Найдите и откройте диск C:\
   - Найдите папку "КООПЕРАНТ", созданную в шаге 1.1
   - Выделите папку "КООПЕРАНТ"
2. Нажмите кнопку "Выбрать папку" или "Разрешить доступ"
3. Подтвердите доступ к папке в диалоговом окне браузера

### 3.3 Создание структуры папок
1. Система автоматически создаст подпапки:
   - Contracts (для договоров)
   - Reports (для отчетов)
   - Payments (для платежных документов)
   - Other (для прочих документов)
2. Дождитесь сообщения: "Директория для хранения документов успешно настроена!"

## 4. Загрузка начальных данных

### 4.1 Использование демо-данных
1. В корне папки системы найдите файл `demo_data.json`
2. В системе нажмите любую кнопку, которая может вызвать загрузку данных
3. Выберите файл `demo_data.json` для загрузки демо-данных

### 4.2 Проверка работы системы
1. Перейдите в раздел "Пайщики" и проверьте наличие демо-данных
2. Перейдите в раздел "Паевые взносы" и проверьте наличие взносов
3. Перейдите в раздел "Бухгалтерия" и проверьте наличие проводок
4. Перейдите в раздел "Документы" и проверьте наличие документов

## 5. Тестирование функционала

### 5.1 Тестирование учета пайщиков
1. Перейдите в раздел "Пайщики"
2. Нажмите "Добавить пайщика"
3. Заполните поля и нажмите "Сохранить"
4. Убедитесь, что пайщик появился в списке

### 5.2 Тестирование паевых взносов
1. Перейдите в раздел "Паевые взносы"
2. Нажмите "Добавить взнос"
3. Заполните данные и нажмите "Сохранить"
4. Убедитесь, что взнос появился в списке

### 5.3 Тестирование документов
1. Перейдите в раздел "Документы"
2. Нажмите "Загрузить документ"
3. Выберите любой тестовый файл (PDF, DOC, JPG и т.д.)
4. Укажите название и тип документа
5. Нажмите "Загрузить"
6. Убедитесь, что документ появился в списке
7. Нажмите "Просмотр" и проверьте кнопки "Открыть файл" и "Скачать"

## 6. Настройка безопасности

### 6.1 Защита папки C:\КООПЕРАНТ
1. Щелкните правой кнопкой по папке C:\КООПЕРАНТ
2. Выберите "Свойства"
3. Перейдите на вкладку "Безопасность"
4. Настройте права доступа по необходимости

### 6.2 Настройка браузера
1. В настройках браузера найдите раздел "Конфиденциальность и безопасность"
2. Убедитесь, что разрешено использование File System API для локальных файлов
3. Добавьте папку с системой в доверенные источники (если используется антивирус)

## 7. Резервное копирование

### 7.1 Создание резервной копии данных
1. Регулярно создавайте копии файлов данных системы
2. Копируйте папку C:\КООПЕРАНТ для резервного хранения документов
3. Храните резервные копии в безопасном месте

## 8. Завершение настройки

После выполнения всех шагов система будет полностью настроена для работы:
- Папка C:\КООПЕРАНТ настроена для хранения документов
- Структура подпапок создана
- Система готова к вводу данных
- Все функции протестированы

## 9. Возможные проблемы и решения

### 9.1 Не удается выбрать папку C:\КООПЕРАНТ
- Убедитесь, что папка создана правильно
- Проверьте, что вы используете поддерживаемый браузер
- Попробуйте запустить браузер от имени администратора

### 9.2 Документы не сохраняются в папке
- Проверьте права доступа к папке C:\КООПЕРАНТ
- Убедитесь, что браузер имеет разрешение на доступ к файловой системе
- Перезапустите браузер и повторите настройку

### 9.3 Файлы не открываются
- Убедитесь, что на компьютере установлены программы для открытия соответствующих типов файлов
- Проверьте, не заблокированы ли файлы антивирусом

## 10. Поддержка

Если у вас возникли проблемы с настройкой:
1. Проверьте файлы документации в папке системы
2. Ознакомьтесь с файлом `error_guide.md`
3. Обратитесь к файлу `document_storage_manual.md` для дополнительной информации
4. При необходимости свяжитесь с технической поддержкой
