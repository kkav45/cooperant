# Инструкция по использованию новых функций системы

## 1. Внесение паевого взноса имуществом

### 1.1 Добавление имущественного взноса
1. Перейдите в раздел "Паевые взносы"
2. Нажмите кнопку "Добавить взнос"
3. Выберите пайщика из списка
4. Укажите тип взноса
5. В поле "Форма оплаты" выберите "Имущество"
6. Поле "Сумма" станет скрытым, появится поле "Описание имущества"
7. Введите описание имущества в соответствующее поле
8. Укажите дату взноса
9. При необходимости добавьте описание
10. Нажмите "Сохранить"

> **Важно**: После добавления имущественного взноса он будет отображаться как "Не оценено" до проведения оценки.

### 1.2 Оценка имущественного взноса
1. В таблице паевых взносов найдите имущественный взнос со статусом "Не оценено"
2. Нажмите кнопку "Оценить" в строке взноса
3. В открывшейся форме вы увидите описание имущества
4. Укажите дату оценки (по умолчанию сегодняшняя дата)
5. Введите оценочную стоимость в поле "Оценочная стоимость"
6. Выберите метод оценки (рыночная стоимость, балансовая стоимость, экспертная оценка и др.)
7. При необходимости укажите ФИО оценщика
8. Добавьте примечания к оценке
9. Нажмите "Оформить акт оценки"

После оценки сумма взноса обновится, и он будет отображаться с указанной стоимостью.

## 2. Выбытие пайщика

### 2.1 Инициация процедуры выбытия
1. Перейдите в раздел "Пайщики"
2. Найдите активного пайщика (со статусом "Активный")
3. В строке пайщика нажмите кнопку "Выбытие"
4. В открывшейся форме укажите:
   - Причину выбытия (из выпадающего списка)
   - Дату выбытия (по умолчанию сегодняшняя дата)
   - Примечания (по желанию)
5. Нажмите "Подтвердить выбытие"

> **Важно**: Нельзя исключить пайщика, если у него есть неоплаченные взносы. Сначала необходимо погасить задолженность.

### 2.2 Возврат паевого взноса при выбытии
После подтверждения выбытия автоматически откроется форма возврата паевого взноса:
1. В поле "Сумма к возврату" будет автоматически рассчитана сумма
2. Выберите форму возврата:
   - Наличные
   - Безналичные
   - Имущество
3. Укажите дату возврата
4. При выборе формы "Имущество" появится поле для описания возвращаемого имущества
5. Укажите номер документа (акт возврата паевого взноса)
6. При необходимости отредактируйте описание операции
7. Нажмите "Выполнить возврат"

После этого пайщик будет переведен в статус "Выбыл", и будет оформлен возврат паевого взноса.

## 3. Восстановление членства

### 3.1 Восстановление пайщика
1. Перейдите в раздел "Пайщики"
2. Найдите пайщика с соответствующим статусом (Выбыл, Исключен и т.д.)
3. В строке пайщика нажмите кнопку "Восстановить"
4. Подтвердите действие в появившемся диалоге
5. Пайщик будет переведен в статус "Активный"

## 4. Просмотр и анализ данных

### 4.1 Просмотр имущественных взносов
- В таблице паевых взносов имущественные взносы отображаются с пометкой "Имущество"
- Если взнос не оценен, рядом будет надпись "Не оценено" красным цветом
- При наведении на "Имущество" отображается описание имущества

### 4.2 Просмотр возвратов
- Возвраты паевых взносов отображаются в таблице с типом "Возврат паевого взноса"
- В бухгалтерских проводках возвраты помечаются как "Возврат" с указанием пайщика

### 4.3 Отчеты
- В отчете по паевым взносам теперь отдельно показываются взносы и возвраты
- В бухгалтерском балансе проводки разделены на поступления и возвраты
- В таблице проводок отображается тип операции (поступление или возврат)

## 5. Бухгалтерские проводки

### 5.1 Автоматическое создание проводок
- При внесении паевого взноса автоматически создается проводка:
  - Для денежных взносов: Дт 51 Кт 86
  - Для имущественных взносов: Дт 08 Кт 86
- При возврате паевого взноса создается проводка:
  - При возврате деньгами: Дт 86 Кт 51
  - При возврате имуществом: Дт 86 Кт 08

### 5.2 Связывание проводок
- Каждая проводка связана с соответствующим паевым взносом
- В таблице проводок отображается информация о связанной операции
- При удалении взноса автоматически удаляются связанные проводки

## 6. Рекомендации по использованию

### 6.1 Для бухгалтеров
- Регулярно проверяйте статус имущественных взносов
- Своевременно проводите оценку имущества
- Контролируйте правильность автоматических проводок
- Используйте отчеты для анализа деятельности

### 6.2 Для администраторов
- Контролируйте процесс выбытия пайщиков
- Следите за задолженностями перед выбытием
- Обеспечивайте корректность данных
- Проводите регулярное резервное копирование

### 6.3 Для руководства
- Используйте отчеты для анализа состава и деятельности кооператива
- Мониторьте движение паевых взносов
- Анализируйте причины выбытия пайщиков
- Оценивайте эффективность работы системы
