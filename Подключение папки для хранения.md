# Требования к форме добавления пайщика и заявлению

## Форма "Добавить нового пайщика"

### Обязательные поля (кроме примечания):
1. ФИО (в одном поле)
2. Дата рождения
3. Адрес проживания
4. Контактный телефон
5. Дата вступления (по умолчанию - сегодня)
6. Статус (активный/неактивный)

### Необязательные поля:
- Примечания
- Паспортные данные (не требуются)
- Гражданин не имеет самостоятельного заработка, получает государственные пособия, пенсию или стипендию (чекбокс, компактно с надписью)

### Генерация ID:
- Использовать номер телефона в формате 70000000000

### Дополнительная информация:
- В окне добавления пайщика-физического лица должны быть три кнопки: "Сохранить", "Отмена", "Добавить взнос".
- При нажатии "Сохранить" - данные пайщика сохраняются, окно закрывается.
- При нажатии "Отмена" - окно закрывается, данные не сохраняются.
- При нажатии "Добавить взнос" - автоматически сохраняются данные пайщика и открывается форма для добавления паевого взноса для этого пайщика.
- Все кнопки должны быть оформлены в едином стиле.
- В форме "Добавить паевой взнос" кнопки должны быть в едином стиле.
- При нажатии на кнопки "Сохранить" и "Отмена" в форме "Добавить паевой взнос" запись должна сохраняться или отменяться соответственно.
- Проанализировать все формы и окна, где используются кнопки. Сделать все кнопки в едином стиле и сохранить это.
- После регистрации пайщика-физического лица должен автоматически формироваться печатный документ - удостоверение пайщика.
- В колонке "Действия" таблицы пайщиков добавить кнопку "Посмотреть" в виде глаза.
- При нажатии на кнопку "Посмотреть" открывается правое боковое меню с информацией о пайщике.
- В боковом меню должна быть кнопка для получения ссылки на скачивание PDF-документа удостоверения пайщика.
- При регистрации пайщика-физического лица удостоверение должно автоматически генерироваться и появляться в боковом окне "Посмотреть".

## Форма заявления для юридического лица

### Обязательные данные юридического лица:
1. Наименование организации
2. Место нахождения (юридический адрес)
3. Государственный регистрационный номер (ОГРН)
4. Идентификационный номер налогоплательщика (ИНН)
5. КПП (при наличии)
6. Банковские реквизиты:
   - Наименование банка
   - БИК банка
   - Корреспондентский счёт
   - Расчётный счёт
   - Назначение платежа

### Данные представителя юридического лица:
1. ФИО
2. Должность
3. Контактный телефон
4. Email
5. Документ, удостоверяющий личность
6. Доверенность (номер, дата выдачи, кем выдана)

### Обязательные поля для юрлица:
- Все вышеуказанные данные юрлица
- Все данные представителя
- Дата подачи заявления (по умолчанию - сегодня)

## Общие требования

### Календарь:
- Дата по умолчанию - сегодня

### Нумерация документов:
- Номер для документа проставляется автоматически

### Интерфейс:
- Вместо модальных окон использовать боковое правое меню шириной 40%
