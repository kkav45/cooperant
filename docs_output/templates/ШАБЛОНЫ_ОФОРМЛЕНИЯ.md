## 1. ОБЩИЕ ТРЕБОВАНИЯ К ОФОРМЛЕНИЮ

### 1.1 Параметры страницы

```
┌─────────────────────────────────────────────────────────┐
│                    297 мм (A4)                          │
│  ┌───────────────────────────────────────────────────┐  │
│  │ 20 мм │                                   │ 10 мм │  │
│  │     ┌─┴───────────────────────────────────┐       │  │
│  │     │                                     │       │  │
│  │     │           ОСНОВНАЯ                  │       │  │
│  │     │           ОБЛАСТЬ                   │       │  │
│  │ 15  │                                     │  10   │  │
│  │ мм  │           (текст документа)         │  мм    │  │
│  │     │                                     │       │  │
│  │     │                                     │       │  │
│  │     └─────────────────────────────────────┘       │  │
│  │ 20 мм                                             │  │
│  └───────────────────────────────────────────────────┘  │
└─────────────────────────────────────────────────────────┘
```

**Поля:**
- Левое: 20 мм
- Правое: 10 мм
- Верхнее: 15 мм
- Нижнее: 20 мм

**Размер листа:** A4 (210 × 297 мм)

### 1.2 Шрифты

| Элемент | Шрифт | Размер | Начертание |
|---------|-------|--------|------------|
| Основной текст | Times New Roman | 14 pt | Обычный |
| Заголовки 1 уровня | Times New Roman | 16 pt | Полужирный |
| Заголовки 2 уровня | Times New Roman | 14 pt | Полужирный |
| Заголовки 3 уровня | Times New Roman | 14 pt | Обычный |
| Таблицы | Times New Roman | 12 pt | Обычный |
| Примечания | Times New Roman | 12 pt | Обычный |

**Межстрочный интервал:** 1,5

### 1.3 Нумерация страниц

- Нумерация сквозная, арабскими цифрами
- Номер страницы проставляется внизу по центру
- Титульный лист включается в нумерацию, но номер не проставляется
- Приложения нумеруются отдельно (А, Б, В...)

### 1.4 Структура документа по ГОСТ 19

```
1. ТИТУЛЬНЫЙ ЛИСТ
2. ЛИСТ УТВЕРЖДЕНИЯ
3. ЛИСТ ИЗМЕНЕНИЙ
4. СОДЕРЖАНИЕ
5. ОСНОВНАЯ ЧАСТЬ
   5.1. Введение
   5.2. Основная часть (разделы по теме документа)
   5.3. Заключение (при необходимости)
6. ПРИЛОЖЕНИЯ
7. ЛИСТ РЕГИСТРАЦИИ ИЗМЕНЕНИЙ
```

## 4. ПРАВИЛА ОФОРМЛЕНИЯ ТАБЛИЦ

### 4.1 Общие требования

```
                         Таблица 1 — [Наименование таблицы]

┌─────────┬──────────────────┬───────────────┬───────────────┬─────────┐
│  № п/п  │  Наименование   │   Ед. изм.    │   Количество  │Примечание│
│         │    показателя   │               │               │          │
├─────────┼──────────────────┼───────────────┼───────────────┼─────────┤
│    1    │       2          │      3        │       4       │    5     │
├─────────┼──────────────────┼───────────────┼───────────────┼─────────┤
│         │                  │               │               │          │
└─────────┴──────────────────┴───────────────┴───────────────┴─────────┘
```

**Требования:**
- Таблица нумеруется арабскими цифрами
- Наименование таблицы располагается над таблицей
- Таблица выравнивается по ширине страницы
- Шрифт в таблице — 12 pt

### 4.2 Пример заполнения

```
                         Таблица 1 — Модули системы

┌─────┬──────────────────┬───────────────┬───────────────┬─────────┐
│ №   │  Наименование    │   Назначение  │   Статус      │Примечание│
├─────┼──────────────────┼───────────────┼───────────────┼─────────┤
│  1  │  Пайщики         │  Управление   │    ✅         │  Готов   │
│     │                  │  данными      │               │          │
├─────┼──────────────────┼───────────────┼───────────────┼─────────┤
│  2  │  Взносы          │  Учёт взносов  │    ✅         │  Готов   │
└─────┴──────────────────┴───────────────┴───────────────┴─────────┘

```
