# Сводный перечень всех функций системы

## 1. Функции учета пайщиков

### 1.1 Добавление пайщика
- Функция: addMember()
- Описание: Добавляет нового пайщика в систему
- Входные параметры: ФИО, статус, дата вступления, контакт, адрес, примечания
- Выходные данные: Новый объект пайщика
- Тип: CRUD операция (Create)

### 1.2 Редактирование пайщика
- Функция: editMember(id)
- Описание: Открывает форму редактирования пайщика
- Входные параметры: ID пайщика
- Выходные данные: Модальное окно с формой редактирования
- Тип: CRUD операция (Update)

### 1.3 Просмотр пайщика
- Функция: viewMember(id)
- Описание: Отображает детали пайщика в боковом меню
- Входные параметры: ID пайщика
- Выходные данные: Боковое меню с информацией о пайщике
- Тип: CRUD операция (Read)

### 1.4 Удаление пайщика
- Функция: deleteMember(id)
- Описание: Удаляет пайщика из системы
- Входные параметры: ID пайщика
- Выходные данные: Результат удаления
- Тип: CRUD операция (Delete)

### 1.5 Поиск пайщиков
- Функция: searchMembers()
- Описание: Поиск пайщиков по ФИО или контактной информации
- Входные параметры: Поисковый запрос
- Выходные данные: Отфильтрованный список пайщиков
- Тип: Поиск

### 1.6 Выбытие пайщика
- Функция: withdrawMember(memberId)
- Описание: Процесс исключения пайщика с возвратом паевого взноса
- Входные параметры: ID пайщика
- Выходные данные: Форма возврата паевого взноса
- Тип: Бизнес-процесс

### 1.7 Восстановление членства
- Функция: restoreMember(memberId)
- Описание: Восстановление членства выбывшего пайщика
- Входные параметры: ID пайщика
- Выходные данные: Обновленный статус пайщика
- Тип: Бизнес-процесс
